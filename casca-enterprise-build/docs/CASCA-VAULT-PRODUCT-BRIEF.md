## 什麼是 Casca Vault？

Casca Vault 是 Casca AI 路由引擎的**企業自建版本**。

你的團隊每天送出成千上萬筆 AI 請求給 GPT-4o。但其中 60% 的問題用便宜模型就能回答得一樣好。Casca Vault 自動判斷每筆請求的複雜度，把簡單問題送便宜模型、複雜問題送頂級模型，**在不犧牲品質的前提下，大幅降低你的 LLM 帳單。**

與 Casca 雲端版不同，**Vault 完整運行在你自己的伺服器上**。你的 prompt、資料、API Key 永遠不離開你的網路。

## 怎麼整合？

**改一行程式碼**：

```python
# 改之前
client = OpenAI(api_key="sk-xxx")

# 改之後
client = OpenAI(
    api_key="sk-xxx",
    base_url="http://你的伺服器:3001/v1"  # 指向 Casca Vault
)
```

就這樣。不用改 prompt、不用改邏輯、不用改任何業務程式碼。

## 部署架構

```
你的伺服器（Docker Compose 一鍵啟動）
┌─────────────────────────────────────────────┐
│                                              │
│  Casca Engine  ←→  MiniLM Classifier         │
│  (API 閘道)        (補強判斷)                │
│       ↕                                      │
│  PostgreSQL    ←→  Casca Agent               │
│  (本地日誌)        (授權 + 用量回報)          │
│                                              │
│  你的 LLM Key → OpenAI / Anthropic / Google  │
└─────────────────────────────────────────────┘
```

**最低需求**：4 CPU · 8GB RAM · 50GB SSD · Docker
